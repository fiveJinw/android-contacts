# android-contacts - 간단한 연락처 구현(1주차)
## 구현할 기능 목록

**지정된 형식의 데이터를 입력하는 기능**
- [x]  문자열을 입력할 수 있는 기능
- [x]  숫자만 입력할 수 있는 기능
- [ ]  날짜 선택 기능
- [ ]  성별 선택 기능

**더보기 기능**
- [ ]  더보기를 클릭하면 숨겨져 있던 입력 폼을 드러낼 수 있는 기능

**저장 기능**
- [ ]  저장 버튼을 클릭하면 토스트 메세지를 출력
- [ ]  이름, 전화번호의 칸이 채워져 있지 않으면 저장이 되지 않으면 그에 맞는 토스트 메세지를 출력

**취소 기능**
- [ ]  취소 버튼을 클릭하면 토스트 메세지를 출력
